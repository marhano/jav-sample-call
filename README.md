## dependency
- JDK21
- Gradle v8.5
- IDE (you can use vscode)

## how to run
### vs code
install Java, Language Support for Java(TM) by Red Hat extention in vs code.
in the terminal run

``` 
./gradlew clean build
./gradlew run
```
### eclipse
